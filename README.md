## Description
Jдностраничное приложение, которое при POST запросе проверяет, является ли переданный ОГРН или ИНН субъектом малого или среднего предпринимательства на сайте https://rmsp.nalog.ru/

## Technical specifications
 - python 3.8.1;
 - fastAPI 0.54.2;
 - VueJS;
## Getting started
 - `python main.py`
 - `cd frontend`
 - `npm install`
 - `npm run server`
